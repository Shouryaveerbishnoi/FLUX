# FLUX

Career development and interview preparation platform.

## Project Structure

```
FLUX/
├── backend/          - Python Flask/FastAPI backend
├── frontend/         - HTML/CSS/JavaScript frontend
└── README.md         - This file
```

## Getting Started

### Backend Setup
```bash
cd backend
pip install -r requirements.txt
python app.py
```

### Frontend
Open `frontend/index.html` in your browser.

## Features

- Profile management
- Career tracking
- Mentor assistance
- Interview preparation
- Progress monitoring
